## MoneyGram Partnership Model

### Why MoneyGram?

- **500,000+ locations** in 180+ countries
- **Already integrated** with Stellar network
- **Licensed and Regulated** globally for cash handling
- **Proven Infrastructure** for cash-to-USDC conversion

### How Stellarama Adds Value

MoneyGram provides the **rails** (Cash ↔ USDC), while Stellarama provides the **experience**.

- **MoneyGram:** Handles regulatory compliance, cash custody, and on/off-ramps.
- **Stellarama:** Provides a seamless mobile interface, instant settlement logic, and Gulf-India corridor specialization.

### Production Flow

1.  **Deposit:** User deposits Cash AED at MoneyGram Dubai → Receives USDC in Stellarama.
2.  **Send:** User sends USDC instantly to family via Stellarama app.
3.  **Withdraw:** Family withdraws Cash INR at MoneyGram Mumbai.

### Revenue Model

| Party          | Fee Structure                     |
| :------------- | :-------------------------------- |
| **MoneyGram**  | ~2% total (Deposit + Withdrawal)  |
| **Stellarama**   | 0.5% Transfer Fee                 |
| **Total Cost** | **~2.5%** (vs Western Union 6-7%) |

**User saves ~60% compared to traditional methods.**
